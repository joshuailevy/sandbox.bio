Our goal is to analyze wastewater hybrid capture data using standard bioinformatics tools.

Hybrid capture is a sequencing method that allows for the screening of a number of target pathogens in a single sequencing run using a pre-defined probe set. It has the advantage of filtering out unwanted organisms, such as plant viruses and benign bacteria, which typically comprise the majority of a metagenomic sequencing run. It also allows for the continual analysis of multiple pathogens of interest over time.

The data we will be using is from [Child et. al (2023)](https://doi.org/10.1128/mbio.01468-23). Here, the group used the Respiritory Virus Oligos Panel (RVOP) to perform hybrid capture sequencing for respiratory viruses in wastewater samples collected in the UK.
