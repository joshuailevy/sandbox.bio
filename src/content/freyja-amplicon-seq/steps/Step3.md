For the rest of this tutorial, we'll be using Github Codespaces to run Freyja.

1. Navigate to <Link href="https://github.com/dylanpilz/freyja-tutorial">https://github.com/dylanpilz/freyja-tutorial</Link>
2. Under the green "Code" button, click the "Codespaces" tab.
3. Click the "WES 2026" codespace 
4. Wait for codespaces to initialize, this may take a few minutes.
5. Begin the next stage by opening README.md using the file explorer.
