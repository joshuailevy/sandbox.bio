For the rest of this tutorial, we'll be using Github Codespaces to run iVar and Freyja.

1. Make a free Github account if you don't already have one.
2. Navigate to <Link href="https://github.com/dylanpilz/freyja-tutorial">https://github.com/dylanpilz/freyja-tutorial</Link>
3. Under the green "Code" button, click the "Codespaces" tab.
4. Click the "WES 2026" codespace 
5. Wait for codespaces to initialize, this may take a few minutes.
6. Begin the next stage by opening README.md using the file explorer.
