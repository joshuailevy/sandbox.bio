Here, we will be showing how to use `ivar` and `freyja` to analyze amplicon sequencing data for mpox.

Briefly, amplicon sequencing relies on pre-defined primer sequences to amplify specific genomic regions of interest for a given pathogen. This lends itself to increased precision and sensitivity for a target pathogen, making it the method of choice for routine surveillance of pathogens and detecting novel variants. 

The data we will be using was generated using the amplicon sequencing read simulator tool [Bygul](https://github.com/andersen-lab/Bygul) using mpox clinical genomes.
